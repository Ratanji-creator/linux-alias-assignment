# FileManager Assignment

## Description
This is a Bash script that performs basic file and directory operations in Linux.

## Directory Operations
- addDir
- deleteDir
- listFiles
- listDirs
- listAll

## File Operations
- addFile
- addContentToFile
- addContentToFileBegining
- showFileBeginingContent
- showFileEndContent
- showFileContentAtLine
- showFileContentForLineRange
- moveFile
- copyFile
- clearFileContent
- deleteFile

## Usage
./FileManager.sh addDir /tmp dir1
./FileManager.sh listFiles /tmp
